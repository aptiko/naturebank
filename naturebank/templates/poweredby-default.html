Ο Ιστοχώρος - Βάση δεδομένων NatureBank αποτελεί ελεύθερο λογισμικό και βασίζεται αποκλειστικά σε τεχνολογίες ελεύθερου λογισμικού. Το λογισμικό NatureBank μπορείτε να τα αποκτήσετε από τη διεύθυνση https://github.com/ellak-monades-aristeias/naturebank.

Η NatureBank έχει αναπτυχθεί σε Python σε συνδυασμό με το περιβάλλον ανάπτυξης εφαρμογών Web Django καθώς και GeoDjango για την ανάκτηση των γεωγραφικών δεδομένων. Η απεικόνιση των γεωγραφικών δεδομένων στο περιβάλλον του χρήστη γίνεται με OpenLayers. Οι χάρτες παρέχονται ελεύθερα από το OpenStreetMap. Η επεξεργασία των γραφικών του ιστοχώρου έγινε με την εφαρμογή GIMP. Η NatureBank έχει δοκιμαστεί και συστήνεται να εγκατασταθεί σε λειτουργικό σύστημα GNU/Linux (διανομή Debian), web server Apache και σχεσιακή βάση δεδομένων (RDBMS) PostgreSQL. Η επέκταση PostGIS της PostgreSQL είναι το σχεσιακό μοντέλο για τα γεωγραφικά δεδομένα.
